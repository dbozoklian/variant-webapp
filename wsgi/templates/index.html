ABraOM

This databank represents a survey of the genetic diversity of Brazil. Variants were called from 600 exomes across a healthy cohort from Sao Paulo.

We offer access to this data via the web form above or programmatically using a REST API. Access is rate limited. Please see the terms of use for more information.
